# mevn-stack-todo-list

## Fullstack Todo App with Vuejs 3 & Nodejs & Express & Mongodb Database

### Install npm
cd server / 
cd client </br>
npm install </br>
### Run Server Side
 npm run dev
## Run Client Side
npm run serve
